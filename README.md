# 👼 INS-MV 3 Character Generator 😈

Frontend tool for the magnificent and delightfully infamous French tabletop RPG
**In Nomine Satanis / Magna Veritas (INS-MV), 3rd edition**

A lightweight web-based project designed to support character creation and generation for INS-MV 3rd edition, fully running in the browser without backend dependency.

---

## 🎲 Current Status (WIP)

- Random NPC generator (early development stage)

---

## 🔧 Project Direction

This project is intended to evolve into a lightweight INS-MV 3rd edition character tool composed of two independent systems:

- a random NPC generator for fast character creation
- a custom, step-by-step character builder for more detailed character design

Both systems share a common foundation but will remain functionally distinct.

---

## 🎯 Objectives

The project aims to progressively build:

- a structured, data-driven generation system based on game rules
- a modular and maintainable frontend architecture (Vanilla JavaScript)
- a flexible foundation supporting multiple character creation workflows
- a clean separation between random generation and guided creation

The current focus is on establishing the core logic and data model before expanding features and UI complexity.

---

## 🧱 Technical Stack

- HTML / CSS / JavaScript (ES6+ Vanilla)
- Static JSON data files as the core database
- Fully frontend-based (no backend)
- Compatible with GitHub Pages deployment

---

## 📂 Data-Driven Architecture

The system aims to be built around modular JSON datasets, including:

- character archetypes and superiors
- talents and abilities
- powers and limitations
- equipment
- demographic and naming data
- incarnation generation data

This structure will allow for easy extension without modifying core logic.
